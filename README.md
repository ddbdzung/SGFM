# Simple Group Finance Management

## Require:

### Nodejs v18.13.0
### MongoDB v6.0.3
### NPM v8.19.2

## Install:

`npm i -g pnpm`

`pnpm install`

Config env file: Remove .template extension path in .env.template file, contact host project for more infomation

`pnpm run dev`
